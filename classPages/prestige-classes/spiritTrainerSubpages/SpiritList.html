The Spirit List
This is a list of the known Spirits, their Stats, Default Moves, and Onboard Effects.
Included here is some information to help understand the statistics involved in handling Spirits:

Spirit Level - A spiritâ€™s level is always considered to be equal to their trainer's level (Spirit Trainer Levels count as 2).
EXAMPLE: Sarah the Spirit Trainer is a level 3 rogue and level 4 Spirit Trainer. Her spirits are effectively level 11. (3 + (4*2) = 11)

Spirit Core Stats - Every Spirit has four core stats. Each time their trainer gains a Spirit Trainer level (including level 1), increase one of their four core stats by 1. When the Trainer bonds with a new spirit, add a number of stat points equal to their Spirit Trainer level.
A brief description of each of the core stats:

  1. Aggression [Agg] - Used for Attack Rolls and CMB rolls made by some offensive Moves.
  2. Resilience [Res] - Used for AC, CMD, DR, Saves, and some defensive Moves.
  3. Collaboration [Col] - Used for Assist Rolls and most Onboard Effects.
  4. Presence [Pre] - Used for DCs and for some Moves.



Spirit Moves - Every Spirit knows 4 Moves they can use in battle.
Each Move has a couple components:

  1. Type - How the Move is determined to be a success or a failure. Attack Roll, Save to Negate, Support, and Assist Roll
  2. Effect - A description of what the Move actually does.
  3. Range - How far from the Spirit the target can be. 'Touch' means the Spirit must reach the target and '-' means the Move is sourced at the Trainer.
  4. Cooldown - How many turns must pass before the Move can be used again. Cooldown 0 means the Move is ready to use next turn.
  5. Special - Any other notes about how the Move works.
