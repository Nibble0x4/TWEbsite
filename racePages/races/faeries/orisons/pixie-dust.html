Description
School: Transmutation
Tier: 0
Components: Somatic, Material(Dandruff)
Casting Time: 1 Action
Range: Touch
Targets: 1 creature or object touched
Duration: 10 minutes
Saving Throw: Fort Negates




Pixie Dust
Sprinkle magical dust over a creature or object, granting them whimsical traits.

Conjure glittering dust and sprinkle it over an object or creature. This can be used to change the color, texture, taste, or smell of the object. If the object weighs less than 50 pounds, it can be made to levitate or interact with others in a different way.

An example of this may be to make the surface bouncy or make the object slide over surfaces like wet soap. If the creature is Medium or smaller, the dust can effect purely aesthetic changes on them such as hair, skin, and clothing. Additionally, if the creature is unconscious, they can be treated like an object ujnder 50 pounds.
