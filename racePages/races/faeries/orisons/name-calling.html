Description
School: Divination
Tier: 0
Components: Verbal
Casting Time: 1 Action
Range: Medium (100ft. + 10/lvl)
Targets: 1 creature within range
Duration: Instantaneous
Saving Throw: Will Negates




Name Calling
Show a target a glimpse of the Other, blurring what is and what is elsewhere.

Requires Naming.

Creature who can hear you takes 1d10 + Cha damage. If a creature resists this cantrip, they cannot be targeted by it again for 4 rounds.

Faeries take double damage from this if you use their True Name.
