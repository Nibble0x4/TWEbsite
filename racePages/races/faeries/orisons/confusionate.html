Description
School: Enchantment
Tier: 0
Components: Somatic
Casting Time: 1 Action
Range: Close (25 ft. + 5/2 lvls)
Targets: 1 creature within range
Duration: Instantaneous
Saving Throw: Will Negates




Confusionate
Show a target a glimpse of the Other, blurring what is and what is elsewhere.

Requires Naming.

Target becomes Confused for 1 turn. If a creature resists this cantrip, they cannot be targeted by it again for 1 minute.

Faeries suffer a -6 penalty if you use their True Name.
